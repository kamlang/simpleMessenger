How it works ?


  • First you need to get an API key:

    curl --basic --user username:password {{url_for("api.get_api_key")}}

  • Then you can access the whole api functionality using basic http auth:

    curl -H "Authorization: ${api_key}" /api/endPoint

    Feel free to set you're API key as an environment variable i.e

    export api_key = "YOUR API KEY"
