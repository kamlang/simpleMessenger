How it works ?


  • First you need to get your credentials, this is done by creating a new OAuth client. You can create one by clicking on "Get Access" from the "API" menu above.

  • Once you created your client you have to give it authorization to access your data. Click "My Clients" from the "API" menu. You can then authorize your new client to get access to your data.

  • Once you get your code you can exchange it against a token. Authorization token have a 1 hour life time. A refresh token is also provided, so you can request a fresh token without having to redo all the steps.
